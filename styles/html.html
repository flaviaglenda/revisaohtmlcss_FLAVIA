O que é

HTML?

HTML (Linguagem de Marcação de HiperTexto) é o bloco de construção mais básico da web. Define o significado e a estrutura do conteúdo da web. Outras tecnologias além do HTML geralmente são usadas para descrever a aparência/apresentação (CSS) ou a funcionalidade/comportamento (JavaScript) de uma página da web.

Tags de Listas

Listas Não Ordenadas

Uma lista não ordenada, ou lista com marcadores, é um tipo de lista em HTML onde a ordem dos itens não é importante.
Em vez de números ou letras, cada item da lista é precedido por um marcador, que geralmente é um ponto ou um símbolo.

Exemplo:

  • Item um
  • Item dois
  • Item três
  • Item quatro
  • Item cinco

Listas Ordenadas

Listas ordenadas são listas em HTML em que a ordem dos itens é importante e cada item é numerado sequencialmente. Isso é útil quando você deseja apresentar itens em uma sequência específica, como etapas de um processo, classificações ou instruções numeradas.

  1. Item um
  2. Item dois
  3. Item três
  4. Item quatro
  5. Item cinco

Listas Aninhadas

Listas aninhadas são listas que contêm outras listas dentro delas. Em HTML, isso é feito incluindo uma lista lista não ordenada ou lista ordenada dentro de um item de outra lista. Essa estrutura permite criar subníveis de informações, o que pode ser útil para representar hierarquias ou categorias dentro de uma lista principal.
